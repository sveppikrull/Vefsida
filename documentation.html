Project 1 - Documentation

Project description

In this project I will be making a website using html. The main purpose of this website will be to host the projects that I will be doing for the course VÉL403G at the University of Iceland. Further more I want use this opurtunity to make a website that I can use in the future, as a portfolio that can help me when applying to a school or for a job. Therefor I will be trying my best to make the website look as professional as I can with my limited expirience using html.

Making the website

Since I have no expirience using html I decided to use a template taken from the website https://html5up.net/ to make life easier. At first I chose the template Dimensions as I found the design stunning and the userface was simple. However after having spent some time trying to customize the template I discovered that it was maybe to simple, it lacked templates for different types of pages. Therefor I decided to switched to the template Helios as it had multiple templates for different types of pages and the front page userface was more extensive (had a dropdown feature and more).

Most of the time was spent trying out different things and seeing what bits of code influenced witch part of the site. Once I felt that I knew how things worked I started trimming down the template. It had a lot of things going on at once. My goal was to make the website look professional and simple. So when I felt I had removed all the unnecessary parts I started to perfect the home page. Once I had finished the home page I started working on the other pages. I decided on having the pages: About Me, University of Iceland, Personal Work and Resume.

Subpages

On the About Me page I wrote a few paragraphs about myself and my interest and uploded a picture of me. I spent quite alot of time on trying to get the page to look good. I managed to get a look that I am somewhat happy with but I will probably change it in the future when I've gotten more comfortable html.

For the University of Iceland tab I used a dropdown menu where I will link various courses and the big projects I have done. In the dropdown menu there is a link to this course and in particular this project. Furthermore have I linked a few big projects I have done during my studies.

The Personal Work page is rather empty for the time being. Since I haven't done any big projects on my own time I don't have alot to put on there. So in the meantime I have uploaded a few pictures I have taken over the years. I mainly look at this page as something that I can use in the future.

The Resume tab is just a simple link that pulls up my resume.
